# lista_tarefas_flutter

Este projeto é um app de lista de tarefas, contendo banco de dados SQLITE e com o fontend em flutter.

Personalização:  
* Campo Extra: tipoTarefa 
* Tema: temaEnergetico
* Cor primária: orange 
* Cor secundária: deepOrange

Dificuldade encontrada: o uso de um emulador Android para rodar a aplicação exigiu muitos recursos do computador e tornou o processo mais lento.

## Getting Started

//Necessário Android Studio e Flutter instalados na máquina
flutter pub get
flutter run //É necessário ter um emulador Android instalado e configurado (ex. Pixel 5)
